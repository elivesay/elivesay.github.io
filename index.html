AI/Business Solution

ZipAdviser AI Campaign Setup

An AI-powered solution that simplifies campaign setup by translating plain text requests into targeting options and filters.

Technologies: AI/ML, Natural Language Processing, Campaign Management, UI/UX

  • Plain speech to targeting options translation
  • Streamlined campaign setup process
  • Automated demographic filter selection
  • Enhanced user experience for complex configurations

Technical Implementation

The solution consists of an internal API endpoint that accepts requests from the web app (or other tools in the future) and translates plain text using AI Large Language Models.

Architecture Overview
  • Internal API endpoint accepts plain text queries
  • Leverages LLM (Gemini or OpenAI) to parse natural language into structured data
  • Returns demographic filters in JSON format
  • Filters are used in ZipAdviser and sent to Verble or Looker
  • Supports message_history for conversational interactions
Technology Stack
FastAPI Nomad Gemini API OpenAI JSON RESTful API
Workflow
  1. User enters plain text query in web app
  2. Request is sent to internal API endpoint
  3. Endpoint forwards request to appropriate LLM with system instructions
  4. LLM processes text and extracts demographic filters
  5. JSON response is returned and used to filter ZipAdviser query
